## What is this?

POSEIDON is a pentesting firmware for the M5Stack Cardputer-Adv (ESP32-S3). 90+ features across WiFi, BLE, sub-GHz, 2.4 GHz, LoRa, IR, network attacks, and more. In the same family as Flipper Zero, Bruce, Evil-M5Project, and ESP32Marauder ‚ÄĒ but built around a **real keyboard** with letter mnemonics, typed parameters, and 6 swappable visual themes.

Supports three hardware hats (one at a time):
- **M5Stack CAP-LoRa1262** ‚ÄĒ LoRa (SX1262) + GNSS (GPS)
- **PINGEQUA Hydra RF Cap 424** ‚ÄĒ CC1101 sub-GHz + nRF24L01+ 2.4 GHz
- **ESP32-C5 companion node** ‚ÄĒ 5 GHz WiFi + Zigbee via ESP-NOW mesh

## Quick Start

```bash
# Build from source
git clone https://github.com/GeneralDussDuss/poseidon.git
cd poseidon
pio run -t upload

# Or flash the pre-built binary
esptool.py --chip esp32s3 write_flash 0x0 poseidon-v0.3.0-cardputer-adv.bin
```
