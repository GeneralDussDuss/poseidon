What's inside

Six modules behind one menu key. Phase 1 shipped in v0.3; phase 2 lands with v0.4.

DHCP Starvation

Floods the network's DHCP server with random-MAC Discover/Request cycles until the pool exhausts. Live counters. Auto-detects exhaustion on NAK ≥ 20.

PHASE 1

Rogue DHCP (STA + AP)

Races the real server with our own Offer/Ack (STA mode) or is the DHCP server for clients on our SoftAP (AP mode). Gateway + DNS poisoning for chained MitM.

PHASE 2

Responder

LLMNR + NBT-NS name-poisoning plus SMB1 NTLMv2 Type-2 builder. Captured hashes logged to SD in hashcat mode-5600 format. Feed directly to the on-device cracker.

PHASE 2

WPAD NTLM harvest

Serves PAC file on port 80 + HTTP 407 Proxy-Authenticate challenge. Windows clients auto-fetch WPAD, auto-auth with current domain creds.

PHASE 2

On-device NTLMv2 cracker

Pure-C HMAC-MD5 wordlist runner. Reads hashcat-5600 lines from SD, tries each password. No PC needed. Seeds a starter wordlist on first run.

PHASE 2

Custom UX

Own renderer — list / grid / carousel views, RaspyJack-faithful info pages, screensaver, 7-sprite pirate icon pack baked from PNG via tools/sprite_sheet_to_icons.py.

UX

Every file in src/features/saltyjack/ opens with an attribution header crediting @7h30th3r0n3. Authorized testing only — hunt your own LAN.

What's landing next

The two big drops queued behind v0.4. Full ESP32-C5 citizenship, nRF52 companion for Flipper-grade BLE, plus the long-list of features testers have been asking for. Not promises — stuff that's either planned, in-progress, or already prototype-ready.

ESP32-C5 full citizenship

Dual-band 5 GHz + 2.4 GHz WiFi citizen on ESP-NOW mesh. Per-STA deauth (not just per-AP), 5 GHz PMKID + full 4-way handshake capture with EAPOL relay back to Cardputer, 802.15.4 Zigbee PCAP export, Thread network enumeration, multi-C5 coordination from one Cardputer.

v0.5 · HEADLINE

nRF52 companion — Flipper-grade BLE

nRF52840 hat for BLE 5.0 long-range sniffing, LE Secure Connection attacks, full GATT relay, Bluetooth Mesh enumeration, and Flipper-compatible .ir / .sub relay. ESP-NOW link back to Cardputer — same satellite pattern as the C5 node.

v0.5 · HEADLINE

On-device WPA2 cracker

Dual-core PBKDF2-SHA1 runner that eats your captured .22000 files straight from SD. Seeded wordlist, custom dictionary support, resume-from-last-line. No PC tether.

v0.6 · MIMIR

MIMIR drop-box goes live

BPI-M4 Zero server side gets real. hcxdumptool wrapper, Bjorn-style action DAG, FENRIR RL policy head picking exploit strategy. Pull captured .pcap + .22000 back to Cardputer SD over USB-CDC. GPS-tagged attack logging with WiGLE.

v0.6 · DROP-BOX

OTA from SD — one-button updates

Drop a .bin on the SD card root, hit Update in the system menu, dual-partition swap + bootloader handoff. No USB required, no PC required, no losing the device to a brick if the download cuts.

v0.5 · QoL

Thread / Matter enumeration

Via the C5 satellite: passive Thread network discovery, node enumeration, commissioning-invite detection, Matter bridge fingerprinting. Output to SD as structured JSON.

v0.5 · 802.15.4

TRIDENT v1.0 — record + replay

Desktop companion graduates. Lossless MP4 capture of the Cardputer screen, keystroke replay for scripted demos, macOS/Linux installers, screenshot burst mode. No more PyQt6 prerelease vibes.

v0.5 · COMPANION

FENRIR RL autonomous agent

Reinforcement-learning policy head lives in MIMIR and across C5 satellites. Learns which attack sequences land against which target fingerprints, persists Q-table to SD, shares learning between deployed units over ESP-NOW.

v0.6 · AI

CAD-before-TX on LoRa

Channel Activity Detection gate in the Meshtastic transmit path. POSEIDON stops stepping on other nodes' transmissions — go from polite-ish to actually-polite mesh citizen.
